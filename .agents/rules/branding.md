# EUCORA Branding Guidelines

This file enforces the strict branding and identity standards for the **EUCORA** project.

## Identity

- **Application Name**: EUCORA
- **Organization**: BuildWorks.AI
- **Full Branding**: EUCORA by BuildWorks.AI
- **Caption**: End-User Computing Orchestration & Reliability Architecture
- **Footnote**: Build by BuildWorks.AI

## Rules

1.  **Naming**: Always refer to the application as **EUCORA** (all caps).
2.  **Attribution**: When referencing the creator, use **BuildWorks.AI**.
3.  **Logo**: Use `EUCORA-logo2.png` as the source of truth for all visual assets.
4.  **Tone**: Professional, enterprise-grade, "ruthlessly technical".
5.  **License**: Apache License 2.0.

## Enforcement

- All documentation (README, etc.) must prominently display the Branding and Caption.
- All footer notes in generated reports or web UIs must state "Build by BuildWorks.AI".

## Color Palette

Standard BuildWorks.AI / SARAISE branding colors:

-   **Deep Blue (Primary)**: `#1565C0` (Main), `#0D47A1` (Dark)
-   **Gold (Warning)**: `#FF8F00` (Main), `#F57C00` (Dark)
-   **Teal/Electric Blue (Info)**: `#00ACC1` (Main), `#0097A7` (Dark)
-   **Green (Success)**: `#388E3C` (Main), `#2E7D32` (Dark)
